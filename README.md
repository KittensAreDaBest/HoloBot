## Installation
### From Binary
Download `holobot` binary [here](https://github.com/tacticalcatto/holobot/actions).
Create a file called `config.yml`. Inside the file, put your timezone and Discord bot token. Example:
```yml
token: <YOUR_DISCORD_BOT_TOKEN>
timezone: <TIMEZONE>
```
